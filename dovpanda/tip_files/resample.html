Resample¶

Task: How many events happen each hour?

The Old Way¶

bad = df.copy()
bad['day'] = bad.index.date
bad['hour'] = bad.index.hour
(bad
.groupby(['day','hour'])
.count()
)
event_type
day hour
2018-10-09 17 83
18 96
20 91
21 71
22 84
  • Many lines of code
  • unneeded columns
  • Index is not a time anymore
  • missing rows (Did you notice?)

A Better Way¶

df.resample('H').count() # H is for Hour

date_
